# ALGORITHM

## 스터디 진행방식
* 주 2회 진행(화/금)
* 각자 문제 하나씩 가져와 그 중 문제 2개 선택
* 각각 1시간(혹은 40분) 시간 재서 풀기
* 시간이 되면(못 풀어도 중지) 각자 풀이방법 설명

* 공유하고 싶은 문제나 내용은 [여기](https://github.com/algoStudyforCodingTest/ALGORITHM/discussions)
